## University Projects

I have many university projects that I did during college, but since they are very specific, I will not list them here. Some of the projects I did and find worth mentioning are:

### Formula 1 - [SCC0541 - Database Laboratory]

A complete system with queries for a `real database` of Formula 1, with data on races, drivers, teams, etc. We were also provided data on airports around the world and other data to create more complex queries linking all the data. All data were provided as `.csv` files and imported into the database.

The project has a `Server Side Rendering` front-end made with `Flask` and `Jinja2` and a back-end made with `PostgreSQL` using the `pyycopg2` framework. All queries were done in `SQL` and are available in the repository. `Docker` was also used to facilitate the project's execution. Some of the main concepts used for the project considering that the course is a laboratory were `Windows Functions`, `PGSQL`, `Views`, `Triggers`, `Indices`, `Stored Procedures`, and `Functions`.

You can see the source code [here](https://github.com/Ryrden/SCC0541-Formula1/tree/main)

### Caaso Eletroniks - [SCC0560 - Human-Computer Interaction and User Experience]

Front-end of an `Electronics E-commerce` made with `React` and `Typescript` for a UX/UI course. The project's goal was to develop an interface with good `usability` and `user experience` standards, as well as being `responsive` and `accessible`. `Material UI` was used as it follows design guidelines and is easy to use. The site is hosted with `Github Pages`, and you can see it [here](https://ryrden.github.io/SCC-0560-Caaso-Eletroniks/).

You can see the Figma of the project [here](https://www.figma.com/file/AfCh9CNbgu0OakBySHdMjq/Telas-Caaso-Eletroniks?type=design&node-id=0%3A1&mode=design&t=Q1Qnx7jZ4giqqMJ8-1) and the source code [here](https://github.com/Ryrden/SCC-0560-Caaso-Eletroniks)

### Data Structures Exercises - [SCC0502 - Data Structures I]

In the Data Structures I course, various exercises were proposed to practice some of the main data structures, such as `lists`, `stacks`, `queues`, and `trees`. I solved all the proposed exercises and put them in a repository so that others could see and learn from them.

Two assignments were also proposed with the goal of implementing two tree-based data structures, an `AVL tree` and a `B tree`. I implemented both and put them in the same repository as the exercises.

You can see the source code [here](https://github.com/Ryrden/ICMC-SCC0502)
