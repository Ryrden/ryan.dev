### Formula 1 - [SCC0541 - Laboratório de Banco de Dados]

Sistema completo com consultas e queries para um `banco de dados real` da Formula 1, com dados de corridas, pilotos, equipes, etc. também foi fornecido dados de aeroportos do mundo assim como outros dados para que pudessemos criar queries mais complexas relacionandos todos os dados. Todos os dados foram fornecidos como arquivos `.csv` e foram importados para o banco de dados.

O Projeto possui um Front-end no modelo de `Server Side Rendering` feito com `Flask` e `Jinja2` e um Back-end feito com `PostgreSQL` usando o framework `pyycopg2`. Todas as queries foram feitas em `SQL` e estão disponíveis no repositório, foi utilizado também o `Docker` para facilitar a execução do projeto. Alguns dos principais conceitos utilizados para a criação do projeto tendo em vista que a disciplina é de laboratório foram `Windows Functions`, `PGSQL`, `Views`, `Triggers`, `Indíces`, `Stored Procedures` e `Functions`.

Você pode ver o código fonte [aqui](https://github.com/Ryrden/SCC0541-Formula1/tree/main)

### Caaso Eletroniks - [SCC-0560-Caaso-Eletroniks]

Front-end de um `E-commerce de eletrônicos` feito com `React` e `Typescript` para a disciplina de Interação Humano Computador e Experiência do Usuário. O objetivo do projeto foi desenvolver uma interface com bons padrões de `usabilidade` e `experiência do usuário`, além de ser `responsivo` e `acessível`. Foi utilizado o `Material UI` por seguir guidelines de design e ser fácil de utilizar. O site está hospedado com `Github Pages` e você pode ver ele [aqui](https://ryrden.github.io/SCC-0560-Caaso-Eletroniks/).

Você pode ver o Figma do projeto [aqui](https://www.figma.com/file/AfCh9CNbgu0OakBySHdMjq/Telas-Caaso-Eletroniks?type=design&node-id=0%3A1&mode=design&t=Q1Qnx7jZ4giqqMJ8-1) e o código fonte [aqui](https://github.com/Ryrden/SCC-0560-Caaso-Eletroniks)

### Exercícios de Estrutura de Dados - [SCC0502 - Estrutura de Dados I]

Na disciplina de Estrutura de Dados I, foram propostos diversos exercícios para praticar algumas das principais estruturas de dados, como `listas`, `pilhas`, `filas` e `árvores`. Eu resolvi todos os exercícios propostos e coloquei eles em um repositório para que outras pessoas pudessem ver e aprender com eles.

Foi proposto também 2 trabalhos com o objetivo de implementar duas estruturas de dados baseadas em árvores, uma `árvore AVL` e uma `árvore B`. Eu implementei ambas e coloquei no mesmo repositório dos exercícios.

Você pode ver o código fonte [aqui](https://github.com/Ryrden/ICMC-SCC0502)
